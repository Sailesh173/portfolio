# Sailesh Kumar Mishra - Digital Portfolio

A premium, interactive digital resume and portfolio built for Sailesh Kumar Mishra, a Full-Stack Developer specializing in the MERN stack and AI integrations.

## 🚀 Features

- **Cinematic Entrance:** GSAP-powered preloader and hero entrance animations.
- **Three.js Background:** Custom 3D particle swarm that dynamically tracks mouse movement using raycasting physics.
- **Magnetic UI Elements:** Hover interactions that pull buttons and cards towards the cursor with spring-physics.
- **Interactive Neural Core:** A manipulable 3D wireframe core representing AI integration skills.
- **AI Assistant Mockup:** An interactive chat interface that provides quick context about Sailesh's skills.
- **Ambient Audio:** A subtle, toggleable generative audio soundscape built with the Web Audio API.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** Vanilla CSS3 (Custom Properties, Glassmorphism, Responsive Grid/Flexbox)
- **Logic:** Vanilla JavaScript (ES6+)
- **Animations:** GSAP (GreenSock Animation Platform) & ScrollTrigger
- **3D Graphics:** Three.js

## 💻 Running Locally

This project is a static site but utilizes a local server for hot-reloading and avoiding CORS issues with local assets.

1. Ensure you have Node.js installed.
2. Install dependencies (live-server):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. The site will be available at `http://localhost:3000`.
